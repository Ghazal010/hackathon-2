# Todo App Overview

## Purpose
A todo application that evolves from console app to AI chatbot as part of Hackathon II.

## Current Phase
Phase I: In-Memory Python Console App

## Tech Stack
- Python 3.13+
- UV (package manager)
- Claude Code
- Spec-Kit Plus

## Features
- [x] Task CRUD operations (Phase I)
- [ ] User authentication (Phase II)
- [ ] Task filtering and sorting (Phase II)
- [ ] AI chatbot interface (Phase III)
- [ ] Kubernetes deployment (Phase IV)
- [ ] Advanced cloud features (Phase V)

## Hackathon Requirements
- Spec-driven development using Claude Code and Spec-Kit Plus
- No manual code writing - all code must be generated using Claude Code
- Constitution and spec files must be created for every feature
